# Binary-Search-Tree
A binary search tree implementation with the following functions:

## Features
- `buildTree` builds a tree with an array
- `insertNode` inserts a nodedeletes a node
- `findNode` finds the node or null if not found
- `levelOrder` returns an array in level order or performs an action with a callback function
- `preOrder` returns an array in pre order or performs an action with a callback function
- `postOrder` returns an array in post order or performs an action with a callback function
- `inOrder` returns an array in order or performs an action with a callback function
- `height` returns the height of tree
- `depth` returns the depth of node
- `isBalanced` checks if tree is balanced or not
- `rebalanceTree` rebalance tree

  
Demo here https://boriskarl.github.io/binary-search-tree/
